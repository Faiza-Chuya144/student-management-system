# 🎓 Student Management System

A backend-based Student Management System developed using Spring Boot. This system allows managing students, courses, and advising efficiently.

---

## 🚀 Features

- Student registration and management
- Course management
- Advising system (student-course relationship)
- REST API implementation
- Dependency Injection using Spring Boot
- Layered architecture (Controller, Service, Repository)

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring Data JPA / MongoDB
- REST API
- Maven
- JUnit (for testing)

---

## 🧩 Project Structure

- Controller Layer – Handles API requests
- Service Layer – Business logic
- Repository Layer – Database operations
- Model Layer – Entity classes

---

## 🧪 Testing

- Implemented basic unit testing using JUnit
- Tested service layer functionalities
